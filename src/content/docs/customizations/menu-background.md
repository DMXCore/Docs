---
title: Menu Background
---

You can upload an image that will be used as the background of all menus. The file name is `menu-background` and it can be either PNG or JPG. The size of the background is 800×389; however the device will scale any uploaded image to fit. Upload the image in the Web Admin UI under **File Explorer > Images**.
