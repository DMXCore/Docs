## Touchscreen

You can navigate the menu options using either the touch screen, or by rotating and pressing the knob. The general functionality is that a short tap or press on the knob will select the currently highlighted item (black outline). A long-hold will access secondary options for some items — for example, you can edit a cue or preset by long-holding when it's selected. Long-hold will also cancel a popup option.

## Web UI

The Web UI uses standard browser-based navigation:

- **Sidebar** — Click items in the left sidebar to navigate between sections. Expandable groups can be opened by clicking the section header.
- **Breadcrumbs** — A breadcrumb trail at the top of the page shows your current location and lets you navigate back to parent sections.
- **Lists** — Click an item name to open its detail/edit page. Use the action buttons (play, edit, delete) in each row for quick operations.
- **Responsive layout** — The Web UI adapts to different screen sizes. On smaller screens, the sidebar collapses into a hamburger menu.

See [Connecting to the Web UI](/dmx-core-100/getting-started/connecting-to-the-web-ui) for how to access the Web UI.
