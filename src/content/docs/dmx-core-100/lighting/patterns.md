---
title: Patterns
---

Patterns are built-in lighting effects that can be used to quickly test your output configuration without needing to create presets or cues.

![Patterns screen](https://github.com/DMXCore/DmxCore100/assets/407941/77723376-cd7c-408a-a258-58eedd5169e7)

On the **touchscreen**, navigate to **Main Menu > Patterns** and select a pattern to output it immediately. Patterns are primarily a touchscreen feature for quick testing during setup.

For dynamic lighting effects in production, see [Effects](/dmx-core-100/lighting/effects) which provides configurable effect generators that can be applied to specific fixtures and zones.

See [Blackout and Stop](/dmx-core-100/basics/blackout-and-stop) for details about the difference between blackout and stop.
