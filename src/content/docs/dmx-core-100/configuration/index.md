title: Configuration
---

This section covers the configuration options available in the DMX Core 100. Most configuration settings are accessible from both the touchscreen and the Web UI, with the Web UI offering more detailed options.

- [Admin Mode](/dmx-core-100/configuration/admin-mode) — Access protected settings via PIN
- [Settings](/dmx-core-100/configuration/settings) — System, network, and protocol settings
- [Output Config](/dmx-core-100/configuration/output-config) — Configure DMX output universes and protocols
- [Users & Roles](/dmx-core-100/configuration/users-and-roles) — Manage user accounts and permissions
- [Backup & Restore](/dmx-core-100/configuration/backup-and-restore) — Back up and restore your configuration and content
- [Audit Log](/dmx-core-100/configuration/audit-log) — View system activity history
