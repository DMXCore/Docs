This section covers unboxing, physical installation, and first-time setup of the DMX Core 100.

## Choose Your Workflow

Before diving in, it helps to know which workflow fits your use case (see the [overview](/dmx-core-100) for details):

- **Record & Playback** — You have external lighting software and want to record shows as cues for automated playback. After setup, head to [Recording](/dmx-core-100/playback/recording) and [Cues](/dmx-core-100/playback/cues).

- **Fixture Control** — You want to set up fixtures directly in the DMX Core 100 and build scenes with presets and effects. After setup, head to [Lighting](/dmx-core-100/lighting) to configure your fixtures.

Both workflows share the same setup steps below.

## Interfaces

The DMX Core 100 has two interfaces you'll use to set up and operate the device:

- **Touchscreen** — The built-in 4.3" display for direct on-device operation. Start here with [Installation](/dmx-core-100/getting-started/installation), then learn the [Home Screen](/dmx-core-100/getting-started/home-screen) and [Main Menu](/dmx-core-100/getting-started/main-menu).

- **Web UI** — A browser-based interface for full configuration and advanced features. See [Connecting to the Web UI](/dmx-core-100/getting-started/connecting-to-the-web-ui) to access it, then explore the [Web UI Dashboard](/dmx-core-100/getting-started/web-ui-dashboard).
