After logging in, the Web UI displays the Dashboard — your central hub for monitoring and controlling the DMX Core 100.

<!-- SCREENSHOT: Web UI dashboard showing sidebar and main content area (dark mode) -->

## Layout

The Web UI has two main areas:

- **Sidebar** (left) — Navigation menu organized into sections. Click a section header to expand or collapse it.
- **Main content** (right) — The active page. A breadcrumb trail at the top shows your current location.

## Sidebar Navigation

The sidebar is organized into the following sections:

| Section | Contains |
|---------|----------|
| **Dashboard** | System status, currently playing items, playback controls |
| **Lighting** | [Presets](/dmx-core-100/playback/presets), [Effects](/dmx-core-100/lighting/effects), [Cues](/dmx-core-100/playback/cues), [Sounds](/dmx-core-100/playback/sounds), [Timelines](/dmx-core-100/playback/timelines), [Schedules](/dmx-core-100/scheduling-automation/schedules), [Fixture Control](/dmx-core-100/lighting/fixture-control) |
| **Operation** | [Custom Menu](/dmx-core-100/scheduling-automation/custom-menus), [Favorites](/dmx-core-100/scheduling-automation/favorites) |
| **Plugins** | Plugin settings, Q-SYS, Symetrix |
| **Settings** | [Fixtures](/dmx-core-100/lighting/fixture-setup), [Input Triggers](/dmx-core-100/scheduling-automation/input-triggers), [Output Events](/dmx-core-100/scheduling-automation/output-events), [Outputs](/dmx-core-100/configuration/output-config), [Zones](/dmx-core-100/lighting/zones), [Satellites](/dmx-core-100/integrations/satellites), Network, Protocol, System, and more |
| **Utilities** | [Audit Log](/dmx-core-100/configuration/audit-log), [Record](/dmx-core-100/playback/recording), Releases, System info |
| **Backup & Restore** | [Local and cloud backups](/dmx-core-100/configuration/backup-and-restore) |
| **File Explorer** | Browse and manage files on the device |
| **User Management** | [Users & Roles](/dmx-core-100/configuration/users-and-roles) |

:::note
Some sidebar items are only visible to users with the appropriate permissions. If you don't see a menu item, check with your administrator about your role permissions.
:::

## Dashboard Content

The Dashboard shows:

- **Currently playing** items (cues, sounds, timelines) with playback controls (pause, stop, seek)
- **System status** including device temperatures, CPU/memory usage, storage, and network speed
- **Quick actions** for common operations

## Theme

The Web UI supports both light and dark themes. You can switch between them in **Settings > Preferences**.
