---
title: Integrations
---

The DMX Core 100 is built to support different kinds of integrations and triggers. The current integrations are listed below — please contact us to let us work with you on extending this list.

* Open Sound Control (OSC) with popular applications like TouchOSC
* QSC Q-SYS
* Symetrix
* HTTP
* MIDI controllers like Akai LPD8
