# Software

The Pico 2 runs software/firmware that you create yourself using the Raspberry Pi Pico ecosystem. We have provided examples using PicoSDK (C/C++), but it is also possible to use MicroPython.

[Example code](https://github.com/DMXCore/Pico-Examples)

### Commercial Software

We have developed a software suite for the Pico 2 called the [DMX Re-Mapper](https://app.gitbook.com/o/IX1AplGgX61XCtW7kGJr/s/eB1QgQoSPurpEfwtt72b/), which provides features like re-mapping of DMX channels and using digital inputs with a configuration utility. It's possible to purchase this software as an add-on to a standard Pico 2, the firmware is locked to the serial number of the Pico 2. Contact DMX Pro Sales for more information about this upgrade.
